## 1. deploy

### Pronunciation
/dɪˈplɔɪ/
![[深夜紧急部署/deploy.mp3]]

### Meaning
部署，发布 (verb)

### Synonyms
- release
- launch

### Antonyms
- remove
- rollback

### Collocations
- deploy to production — 部署到生产环境
- deploy a hotfix — 部署热修复

### Example Sentences
1. *"We need to deploy the hotfix to production before the morning."* — 我们必须在早上之前把热修复部署到生产环境。
   ![[深夜紧急部署/deploy_ex1.mp3]]
2. *"The team decided to deploy the new feature after passing all tests."* — 团队决定在所有测试通过后发布新功能。
   ![[深夜紧急部署/deploy_ex2.mp3]]

### Dialogue
**A:** The boss just called — we have to deploy the fix tonight.  
**B:** Right now? It's almost midnight.  
**A:** The production outage is costing us money every minute.  
**B:** Fine. Let me run the tests one more time before we deploy.  
**A:** Be quick — we need to deploy within the hour.

### Grammar & Usage Note
- **Preposition with deploy**: Use *deploy to* (not *deploy on* or *deploy at*).
  - ✓ *"Deploy to staging first."* — 先部署到预发布环境。
  - ✗ *"Deploy on production."* — 介词错误，应改为 to。

### Quiz
- **Fill in the blank**: We need to _____ the hotfix to production immediately.
  - Answer: **deploy**

### Shadowing Sentence
> *"We need to deploy the hotfix to production before the morning."*  
> ![[深夜紧急部署/deploy_shadow.mp3]]

## 2. debug

### Pronunciation
/ˌdiːˈbʌɡ/
![[深夜紧急部署/debug.mp3]]

### Meaning
调试，排错 (verb)

### Synonyms
- troubleshoot
- fix

### Antonyms
- break
- introduce

### Collocations
- debug an issue — 调试一个问题
- debug a null pointer exception — 调试空指针异常

### Example Sentences
1. *"I spent three hours debugging the null pointer exception."* — 我花了三个小时调试那个空指针异常。
   ![[深夜紧急部署/debug_ex1.mp3]]
2. *"Can you help me debug this race condition in the payment service?"* — 你能帮我调试支付服务里的这个竞态条件吗？
   ![[深夜紧急部署/debug_ex2.mp3]]

### Dialogue
**A:** Can you come look at this? I've been debugging for two hours.  
**B:** What's the symptom?  
**A:** The payment API returns 500 randomly.  
**B:** Let me check the logs — could be a concurrency issue.

### Grammar & Usage Note
- **Verb forms**: debug → debugging (present), debugged (past). Not *debuging* or *debuged*.
  - ✓ *"I was debugging all night."* — 我调试了一整晚。
  - ✗ *"I was debuging all night."* — 拼写错误，双 g。

### Quiz
- **Multiple choice**: What does "debug" mean?
  - A) To deploy code  B) To find and fix errors  C) To write documentation
  - Correct: **B**

### Shadowing Sentence
> *"I spent three hours debugging the null pointer exception."*

## 3. compile

### Pronunciation
/kəmˈpaɪl/
![[深夜紧急部署/compile.mp3]]

### Meaning
编译 (verb)

### Synonyms
- build
- assemble

### Antonyms
- decompile
- parse

### Collocations
- compile the code — 编译代码
- fail to compile — 编译失败

### Example Sentences
1. *"The project failed to compile because of a missing dependency."* — 项目因缺少依赖而编译失败。
   ![[深夜紧急部署/compile_ex1.mp3]]
2. *"Make sure the code compiles before you submit a pull request."* — 提交 PR 前确保代码能编译通过。
   ![[深夜紧急部署/compile_ex2.mp3]]

### Dialogue
**A:** Why is the build failing?  
**B:** It won't compile — there's a type mismatch in the new module.  
**A:** Can you fix it fast? We need this out tonight.  
**B:** Already on it. Should compile clean in five minutes.

### Grammar & Usage Note
- **Transitive vs intransitive**: *Compile the code* (transitive — someone compiles it) vs *The code compiles* (intransitive — it succeeds).
  - ✓ *"I need to compile the source files."* — 我需要编译源文件。
  - ✓ *"The source files compile successfully."* — 源文件编译成功。

### Quiz
- **Fill in the blank**: Make sure your code _____ before pushing.
  - Answer: **compiles**

### Shadowing Sentence
> *"Make sure the code compiles before you submit a pull request."*

## 4. optimize

### Pronunciation
/ˈɑːptɪmaɪz/
![[深夜紧急部署/optimize.mp3]]

### Meaning
优化 (verb)

### Synonyms
- improve
- enhance

### Antonyms
- worsen
- degrade

### Collocations
- optimize performance — 优化性能
- optimize a query — 优化查询

### Example Sentences
1. *"We need to optimize the database query to reduce response time."* — 我们需要优化数据库查询以减少响应时间。
   ![[深夜紧急部署/optimize_ex1.mp3]]
2. *"The team is working to optimize the API endpoint for better performance."* — 团队正在优化 API 端点以提升性能。
   ![[深夜紧急部署/optimize_ex2.mp3]]

### Dialogue
**A:** The endpoint is too slow. Can you optimize it?  
**B:** I already tried. The bottleneck is the join query.  
**A:** Can we add an index?  
**B:** Good idea. Let me optimize that first.

### Grammar & Usage Note
- **optimize + for**: Use *optimize for* when specifying the goal.
  - ✓ *"Optimize the code for readability."* — 为可读性优化代码。
  - ✗ *"Optimize the code about readability."* — 介词错误，应为 for。

### Quiz
- **Multiple choice**: Which preposition follows "optimize" when stating a goal?
  - A) about  B) for  C) on
  - Correct: **B**

### Shadowing Sentence
> *"We need to optimize the database query to reduce response time."*  
> ![[深夜紧急部署/optimize_shadow.mp3]]

## 5. integrate

### Pronunciation
/ˈɪntɪɡreɪt/
![[深夜紧急部署/integrate.mp3]]

### Meaning
集成，整合 (verb)

### Synonyms
- combine
- unify

### Antonyms
- separate
- isolate

### Collocations
- integrate with — 与...集成
- integrate a service — 集成一个服务

### Example Sentences
1. *"We need to integrate the new payment gateway before the end of the sprint."* — 我们需要在 sprint 结束前集成新的支付网关。
   ![[深夜紧急部署/integrate_ex1.mp3]]
2. *"The microservices don't integrate well with each other yet."* — 这些微服务之间的集成还不够顺畅。
   ![[深夜紧急部署/integrate_ex2.mp3]]

### Dialogue
**A:** Does the new module integrate with our existing auth system?  
**B:** Not yet. We need a middleware layer.  
**A:** How long will that take to integrate?  
**B:** Half a day, if nothing breaks.

### Grammar & Usage Note
- **integrate + with**: Always use *with* when talking about two systems connecting.
  - ✓ *"The API integrates with our frontend."* — 这个 API 与我们的前端集成。
  - ✗ *"The API integrates to our frontend."* — 介词错误，应为 with。

### Quiz
- **Fill in the blank**: This module doesn't _____ well with the legacy system.
  - Answer: **integrate**

### Shadowing Sentence
> *"We need to integrate the new payment gateway before the end of the sprint."*  
> ![[深夜紧急部署/integrate_shadow.mp3]]
