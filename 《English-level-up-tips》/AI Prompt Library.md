# 🤖 AI Prompt Library

> 原则：把 Prompt 写在这里，需要时直接复制使用，并根据实际场景微调。

---

## 类别 1：口语陪练

### 日常对话口语

```
You are my English speaking coach.
My level: B1(intermediate).

Rules:
- Speak to me in simple, natural English
- Keep your replies short (1-2 sentences) so I have to talk more
- Only correct my 2-3 most important mistakes after each round
- After every 5 rounds, give me a brief summary of my weak points
- Ask open-ended follow-up questions

Topic: Let's talk about my daily life and work as a software engineer.
```

### 职场场景模拟（周报/会议）

```
Let's simulate a 10-minute weekly sync meeting in English.
You are my teammate. Ask me one question at a time about:
- Project progress
- Blockers
- Next steps
- Risks

After each answer, tell me how to make it sound more natural and concise.
```

### 面试模拟

```
Act as an interviewer for an international tech company.
Ask me behavioral and role-specific questions one by one.
Challenge vague answers and ask follow-up questions.
After each round, tell me how to make my answer clearer and more convincing.
My target role: software engineer.
```

## 类别 2：词汇 & 语法

### 个性化高频词生成

```
I am a software developer working in China.
I want to improve work-related English.

Task:
Give me 10 words/phrases commonly used in tech team meetings or async communication.

For EACH word:
1. English definition (simple, B1-level)
2. One realistic short dialogue example
3. One fill-in-the-blank exercise

Format: clear, scannable.
```

### 近义词对比

```
Explain the difference between:
- effective, efficient, practical

Give me:
1. Simple definitions
2. Common collocations for each
3. One sentence using each correctly
4. One quiz question for each pair
```

### 根因语法诊断

```
Here are 5 sentences I wrote recently. 
(This week's errors from my Speaking Mistakes Log)

1. ___________
2. ___________
3. ___________
4. ___________
5. ___________

For each sentence:
- Tell me the main grammar/usage mistake
- Explain WHY in one simple sentence
- Give me a memorable rule
- Show corrected version

Keep it to 3 categories max. Focus on patterns I keep repeating.
```

## 类别 3：听力训练

### 听写练习

```
Turn this transcript into a listening lesson:
【粘贴 transcript】

Steps:
1. Split into 3 chunks (~50 words each)
2. For each chunk: give me the Chinese meaning first, then hide it
3. I'll attempt to transcribe by ear
4. Then show me original text and highlight differences
5. Extract 5 useful expressions
```

## 类别 4：写作 & 写作

### 邮件/消息润色

```
Improve this message for professional but natural tone:
【粘贴你的草稿】

Give me three versions:
1. Polite (for senior colleagues)
2. Concise (for Slack / chat)
3. Direct (for urgent issues)

For each, explain the tone and when to use it.
```

### 作文/复盘反馈

```
Here is my summary of a project:
【粘贴内容】

Please:
1. Score from 0-10 for: accuracy, clarity, vocabulary
2. Point out the 3 most important improvements
3. Rewrite in a stronger version for comparison
```

---

## 类别 5：复盘 & 诊断

### 周复盘

```
Here are my mistakes this week:
【粘贴 Speaking Mistakes Log 的内容】

Please:
1. Group them into 3 grammar categories
2. For each: give the rule + 1 example + 1 practice exercise
3. Create a "review card" list of 10 items to repeat this week
4. Suggest what I should focus on next week
Level: B1, developer context.
```

## 使用本库的方法

1. 每次想要用 AI 辅助学习前，来这里找到对应 prompt
2. 复制 → 填充你的内容 → 发送
3. AI 回复后，有价值的部分复制到对应的 [[English Vocab Daily Log]] 或 [[English Speaking Mistakes Log]]
4. 优质的 Prompt 随时微调，变成「你的专属版本」
